# CAPatternMaker
Cellular Automaton Pattern Maker can make pattern and export as jpg, png and svg formats. This is a Processing3 project.
セルオートマトンによって生成した画像をJPEG, PNG, SVG形式に保存することができるアプリケーションです。 Processing3で動作します。
